# Codex Demo Cloud

This repository stores shared working data for Codex across computers.

## Folder Layout

- `permanent/`: Long-term materials that should be kept across projects.
- `codex-cloud/projects/`: Codex cloud-related content grouped by project.
- `docs/`: Repository usage notes and sync rules.

## Storage Rule

When saving new content, choose one of these locations:

- `permanent/`
- `codex-cloud/projects/<project-name>/`

If the destination or project name is unclear, Codex should ask before saving.
