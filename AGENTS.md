# Repository Instructions

This repository is used to synchronize Codex-related working data across computers.

## Required Storage Behavior

Before saving user-provided or generated content, ask which storage area should be used unless the user has already made it clear:

- `permanent/`
- `codex-cloud/projects/<project-name>/`

If the user chooses `codex-cloud/projects/` but does not provide a project name, ask for the project name before saving.

## Folder Purposes

- `permanent/`: Long-term materials that should be kept across projects.
- `codex-cloud/projects/`: Codex cloud-related content, grouped by project.
- `docs/`: Repository documentation, policies, and sync guides.

## Safety

Never store secrets in this repository, including passwords, API keys, tokens, certificates, or private credentials.
